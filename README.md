## Getting Started

### Prerequisites

Need a mysql database with a .env file for login to the database.

* mySql database
* .env - And now also AWS RDS
  ```sh
  DB_URL = "{URL}", DB_USER = "{USER}", DB_PASSWORD = "{PASSWORD}"
  DB_URL_AWS = "{URL}", DB_MASTER_USER = "{USER}", DB_MASTER_PASSWORD = "{PASSWORD}"    
  ```
